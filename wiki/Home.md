# TokenMyBar

Native macOS menu bar app showing live AI token usage for **Claude Code**, **OpenAI Codex**, and **OpenCode** — each vendor's session/weekly windows, reset times, and plan badge, straight from official sources with a local-history fallback.

- [[Installation]] — DMG, Homebrew, build from source
- [[Troubleshooting]] — Keychain prompts, "Sign in" states, Gatekeeper
- [[Development]] — architecture, tests, cutting a release

Requires macOS 14+. MIT licensed.
